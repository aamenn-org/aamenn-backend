🔐 Aamenn Zero-Knowledge Encryption

Best Practice Flow:
• Password is used for both server auth AND local key derivation
• Master Key is generated locally, encrypted with KEK (derived from password)
• Server stores only encrypted master key - can NEVER decrypt it
• All file encryption/decryption happens locally in your browser
📋 Registration Flow:
Password → KDF(password, salt) → KEK → AES-GCM(masterKey, KEK) → encryptedMasterKey → Server

📋 Login Flow:
Server → encryptedMasterKey + salt → KDF(password, salt) → KEK → Decrypt → masterKey (in memory)
